    Anzeige

    TM1637 – Treiber für 4-stellige 7-Segment-LED-Module (MicroPython).

    Verdrahtung

    • CLK → beliebiger GPIO (Push-Pull ok; ideal: Open-Drain mit Pull-Up)
    • DIO → beliebiger GPIO (Push-Pull ok; ideal: Open-Drain mit Pull-Up)
    • VCC → 3.3 V
    • GND → GND

    Funktionen (Auszug)

    • brightness(0..7) – Helligkeit setzen/lesen
    • show("abcd", colon=True) – Text (max. 4) anzeigen, optional mit Doppelpunkt
    • number(n) – Zahl −999..9999
    • hex(0x0000..0xFFFF) – Hexadezimalwert
    • numbers(n1, n2, colon) – Zwei Zahlen mit Doppelpunkt
    • temperature(n) – °C-Anzeige
    • write(segments, pos=0) – Rohsegmente schreiben

    Examples:

    from modules.tm1637 import TM1637
from machine import Pin
disp = TM1637(clk=Pin(16), dio=Pin(17), brightness=3)
disp.show("HAWK", colon=True)
disp.number(123)
disp.hex(0xBEEF)
