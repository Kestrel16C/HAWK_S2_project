    Einführung in Python und MicroPython

    1. Was ist Python?

    Python ist eine universelle, leicht verständliche Programmiersprache mit klarer Syntax.
    Sie wird in vielen Bereichen eingesetzt: Datenanalyse, Webentwicklung, Automatisierung, KI und mehr.

    MicroPython ist eine abgespeckte Version von Python, die speziell für Mikrocontroller wie den Raspberry Pi Pico oder ESP32 entwickelt wurde.
    Sie bietet die wichtigsten Sprachfunktionen von Python, ergänzt um Module für Hardwarezugriff (machine, utime, usw.).

    2. Dateien und Module

    Eine Python-Datei endet auf .py und stellt ein Modul dar.

    # datei: tools.py
def hello(name):
    return f"Hallo {name}!"

    Module werden mit import eingebunden:

    import tools
from tools import hello

    Ein Paket ist ein Ordner mit mehreren Modulen.
    Unter Python wird darin meist eine Datei __init__.py verwendet (in MicroPython optional).

    3. Variablen und Datentypen

    zahl = 42               # int
pi = 3.1415             # float
text = "Hallo Welt"     # str
aktiv = True            # bool
liste = [1, 2, 3]       # list
tupel = (4, 5)          # tuple
daten = {"a": 1}        # dict

    Variablen werden automatisch typisiert.
    Typen sind flexibel und können sich ändern.

    4. Funktionen

    Funktionen werden mit def definiert:

    def addiere(a, b):
    return a + b

ergebnis = addiere(2, 3)
print(ergebnis)

    Standardwerte und Typangaben sind möglich:

    def begruessung(name: str = "Welt") -> str:
    return f"Hallo {name}!"

    5. Klassen und Objekte

    Klassen sind Baupläne für Objekte.

    class Person:
    def __init__(self, name):
        self.name = name

    def sag_hallo(self):
        print(f"Hallo, ich bin {self.name}!")

p = Person("Anna")
p.sag_hallo()

    Vererbung ist ebenfalls möglich:

    class Student(Person):
    def sag_hallo(self):
        print(f"Ich bin Studentin {self.name}.")

    6. Kontrollstrukturen

    if x > 0:
    print("Positiv")

    tue_etwas()

    quadrate = [n*n for n in range(5)]

    7. Fehlerbehandlung

    try:
    ergebnis = 10 / 0
except ZeroDivisionError:
    print("Fehler: Division durch Null!")
finally:
    print("Fertig.")

    8. Dateizugriff

    # Datei schreiben
with open("daten.txt", "w") as f:
    f.write("Hallo Welt!")

# Datei lesen
with open("daten.txt") as f:
    inhalt = f.read()
    print(inhalt)

    In MicroPython liegt das Dateisystem im Flash-Speicher des Mikrocontrollers.

    9. MicroPython-spezifische Grundlagen

    from machine import Pin
import utime

led = Pin(25, Pin.OUT)  # LED-Pin
led.value(1)            # einschalten
utime.sleep(1)
led.value(0)            # ausschalten

    Weitere häufige Module: - machine → Zugriff auf Hardware (GPIO, ADC, PWM, I2C, SPI, UART) - utime → Zeitfunktionen - uos, ujson, ure → abgespeckte Varianten bekannter Python-Module

    10. Projektstruktur

    projekt/
├─ main.py
├─ boot.py              # optional, startet vor main.py
├─ utils.py
└─ README.md

    main.py wird beim Start automatisch ausgeführt.

    11. Tipps

    • Verwende kurze, sprechende Namen.
    • Nutze Funktionen, um Code übersichtlich zu halten.
    • Arbeite modular (mehrere Dateien statt einer großen).
    • MicroPython hat weniger Speicher → sparsam mit Variablen umgehen.
    • Für Hardwaretests: interaktive REPL (z.B. über USB).
